
# Meu Ambiente de Desenvolvimento

Este repositório contém as configurações do meu ambiente de desenvolvimento no Arch Linux, incluindo meu setup para o Neovim, Alacritty, Fish shell, e outras customizações que utilizo no meu dia a dia.

## 🖥️ Configurações Gerais

- **Sistema Operacional:** Arch Linux
- **Terminal:** [Alacritty](https://alacritty.org/)
- **Shell:** [Fish](https://fishshell.com/)
- **Editor de Texto:** [Neovim](https://neovim.io/)
  - Configurações em Lua
- **Gerenciador de Pacotes:** [Pacman](https://archlinux.org/pacman/)

## 📂 Dotfiles

Meus dotfiles estão organizados de forma modular e incluem:

- **Neovim** - Configurações em Lua, com plugins para produtividade.
- **Alacritty** - Tema e mapeamentos de teclas customizados.
- **Fish Shell** - Alias, funções, e prompt personalizado.


## 🖥️ Configuração do Hyprland

Utilizo o ambiente de trabalho Hyprland configurado com o script disponível no repositório ML4W. Este setup oferece uma experiência leve e altamente customizável para gerenciar minhas janelas e fluxos de trabalho.
